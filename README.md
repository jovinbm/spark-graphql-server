# Server

This app is built using Express JS.

## Getting started

### Install dependencies

```bash
yarn
```

### Run the development server:

```bash
yarn run dev
```

## Other Scripts

In the project directory, you can also run:

### `prettier:fix`

```bash
yarn prettier:fix
```

Reformat files according to prettier

### `prettier:lint`

```bash
yarn prettier:lint
```

Lint files according to prettier

### `eslint:fix`

```bash
yarn eslint:fix
```

### `eslint:lint`

```bash

```

### `ts:check`

